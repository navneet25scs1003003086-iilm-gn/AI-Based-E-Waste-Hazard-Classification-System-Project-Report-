# ♻️ AI-Based E-Waste Hazard Classification System

## 📌 Overview

This project uses **Machine Learning** to classify electronic waste (e-waste) as **hazardous or non-hazardous**. It addresses the growing environmental challenges caused by improper e-waste disposal by automating the classification process using AI techniques.

---

## 🎯 Objectives

* Automate e-waste classification
* Reduce human exposure to hazardous materials
* Improve recycling efficiency
* Support sustainable waste management

---

## ⚙️ Methodology

* Data collection and preprocessing
* Feature selection (battery type, condition, wiring exposure, etc.)
* Encoding categorical data using One-Hot Encoding
* Feature scaling using StandardScaler
* Model training using **Logistic Regression**
* Evaluation using standard ML metrics

---

## 📊 Model Performance

* Accuracy: **87%**
* ROC-AUC Score: **0.92+**
* Effective binary classification of hazardous vs non-hazardous waste

---

## 🧠 Technologies Used

* Python
* Pandas, NumPy
* Scikit-learn
* Machine Learning (Logistic Regression)
